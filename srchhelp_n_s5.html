Call Number and Other Number Searches

  • Library of Congress (LC) Call Number
    The main call number used by Rockhurst University and Saint Paul School of Theology.

  • Dewey Call Number
    The main call number used by Avila University, MCC Blue River, MCC Longview, MCC Maple Woods, MCC Penn Valley, Midwestern Baptist Theological Seminary and William Jewell College.

  • Local Call Number
    Locally assigned call number used for audiovisual, curriculum and special collection materials.

  • Government Document (SuDoc & MoDoc) Number
    SuDoc: Superintendent of Documents number used for United States government documents; MoDoc: call number used for Missouri State government documents.

  • Standard Number Search
    International Standard Book Number (ISBN), International Standard Serial Number (ISSN), Music, or other standard number.

  • OCLC Number
    Unique bibliographic control number.
